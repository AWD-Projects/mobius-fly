## 🚀 **QUICK START**

### 1. Install Dependencies

```bash
npm install
# or
yarn install
```

### 2. Configure Environment

Create `.env.local`:

```bash
NEXT_PUBLIC_SITE_URL=http://localhost:3000
NEXT_PUBLIC_SITE_NAME="Mobius Fly"
```

### 3. Update Site Config

Edit `lib/seo/metadata.ts`:

```typescript
export const SITE_CONFIG = {
  url: process.env.NEXT_PUBLIC_SITE_URL || "https://mobiusfly.com",
  // ... other configs
};
```

### 4. Test Locally

```bash
npm run dev
```

Visit:
- http://localhost:3000/robots.txt
- http://localhost:3000/sitemap.xml
- http://localhost:3000/api/og?origin=LAX&destination=JFK

### 5. Deploy to Production

```bash
npm run build
npm run start
```

## 📖 **USAGE EXAMPLES**

### Landing Page

```tsx
// app/page.tsx
import { getLandingMetadata } from '@/lib/seo/metadata';
import { JsonLdMultiple } from '@/components/seo/JsonLd';
import { getOrganizationSchema, getWebSiteSchema } from '@/lib/seo/json-ld';

export const metadata = getLandingMetadata();

export default function Page() {
  return (
    <>
      <JsonLdMultiple
        schemas={[
          getOrganizationSchema(),
          getWebSiteSchema()
        ]}
      />
      <main>...</main>
    </>
  );
}
```

### Flight Detail Page

```tsx
// app/flights/[id]/page.tsx
import { getFlightMetadata } from '@/lib/seo/metadata';
import { getFlightSchema, getBreadcrumbSchema } from '@/lib/seo/json-ld';

export async function generateMetadata({ params }) {
  const flight = await getFlight(params.id);
  return getFlightMetadata({
    origin: flight.origin.code,
    destination: flight.destination.code,
    date: flight.departureDate,
    flightId: flight.id,
  });
}

export default async function FlightPage({ params }) {
  const flight = await getFlight(params.id);

  return (
    <>
      <JsonLd data={getFlightSchema(flight)} />
      <JsonLd data={getBreadcrumbSchema([
        { name: 'Home', url: '/' },
        { name: 'Search', url: '/search' },
        { name: `${flight.origin} to ${flight.destination}`, url: `/flights/${flight.id}` }
      ])} />
      <main>...</main>
    </>
  );
}
```

### Private Page (No Index)

## 🔧 **CUSTOMIZATION**

### Add New Page Type

1. Create metadata generator in `lib/seo/metadata.ts`:

```typescript
export function getNewPageMetadata(params: any): Metadata {
  return {
    title: "Page Title",
    description: "Page description",
    // ... other metadata
  };
}
```

2. Create JSON-LD schema in `lib/seo/json-ld.ts`:

```typescript
export function getNewPageSchema(data: any) {
  return {
    "@context": "https://schema.org",
    "@type": "Thing",
    // ... schema properties
  };
}
```

3. Use in page component:

```tsx
export const metadata = getNewPageMetadata();

export default function NewPage() {
  return (
    <>
      <JsonLd data={getNewPageSchema()} />
      <main>...</main>
    </>
  );
}
```

### Add to Sitemap

Edit `app/sitemap.ts`:

```typescript
const STATIC_PAGES: MetadataRoute.Sitemap = [
  // ... existing pages
  {
    url: `${SITE_CONFIG.url}/new-page`,
    lastModified: new Date(),
    changeFrequency: "weekly",
    priority: 0.8,
  },
];
```

### Block Route from Indexing

Edit `middleware.ts`:

```typescript
const PRIVATE_ROUTE_PATTERNS = [
  // ... existing patterns
  /^\/new-private-route/,
];
```

## 🐛 **COMMON ISSUES**

### Issue: Sitemap not updating

**Cause**: Next.js caching
**Solution**: Clear `.next` cache and rebuild

```bash
rm -rf .next
npm run build
```

### Issue: OG images not showing

**Cause**: Cache or incorrect URL
**Solution**:
1. Test URL directly: `/api/og?origin=LAX&destination=JFK`
2. Clear social media cache
3. Validate image size (1200x630px)

### Issue: Private pages indexed

**Cause**: Missing noindex tag or middleware
**Solution**:
1. Check `middleware.ts` includes route pattern
2. Verify X-Robots-Tag header
3. Request removal in GSC

### Issue: Duplicate content

**Cause**: Missing canonical tags
**Solution**:
1. Add canonical to all pages
2. Use consistent URL format
3. Implement 301 redirects
